2.2 #Mixins.assert-defined assert-defined($name, $description)

[public:{output}] Assert that lookup($name) resolves to a non-null Stylus value. Variables and functions are supported; values intentionally set to null are indistinguishable from missing symbols.

Parameters:

  • $name - {string|ident} Variable or function name to look up
  • $description = null - {string} Optional description emitted in assertion comments.
