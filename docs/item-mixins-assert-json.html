2.5 #Mixins.assert-json assert-json($path, $expected, $description, $options)

[public:{output}] Assert that a JSON fixture loaded through Stylus json() matches the expected hash object. Options default to { hash: true, leave-strings: true }, and hash is always forced to true.

Parameters:

  • $path - {string} JSON file path resolved by Stylus
  • $expected - {object} Expected hash object
  • $description = null - {string} Optional description emitted in assertion comments.
  • $options = null - {object|null} Optional Stylus `json()` options.
